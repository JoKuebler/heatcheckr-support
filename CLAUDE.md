# HeatCheckr Support Website

Static marketing and support site. No build system — plain HTML/CSS.

## Deployment
Hosted on **GitHub Pages** via CNAME-based custom domain (`CNAME` file in repo root).
Deploy by pushing to the git repository.

## Key Files

| File | Purpose |
|------|---------|
| `index.html` | Main landing / support page |
| `style.css` | All styles |
| `404.html` | Custom 404 page |
| `CNAME` | Custom domain config for GitHub Pages |
| `site.webmanifest` | PWA manifest |
| `sitemap.xml` | SEO sitemap |
| `robots.txt` | Crawler rules |
| `favicon.png`, `icon.png` | Site icons |
| `images/` | Screenshot and asset images |

## Theme
- Background: `#0F1115` (dark)
- Accent red: `#C9082A`
- Matches the mobile app's dark aesthetic
